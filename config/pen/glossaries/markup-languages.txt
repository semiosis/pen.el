HCL
HashiCorp configuration language
    https://github.com/hashicorp/hcl

    It's a bit like JSON.

    Examples:
        $HOME/.config/lab.hcl

Creole
    A lightweight common markup language for
    various wikis.

    A light-weight Creole engine is integrated
    in PlantUML to have a standardized way to
    emit styled text.

    https://plantuml.com/creole

    This is **bold**
    This is //italics//
    This is ""monospaced""
    This is --stricken-out--
    This is __underlined__
    This is ~~wave-underlined~~
