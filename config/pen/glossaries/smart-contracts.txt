Berkeley Packet Filter
BPF
    [#linux]
    [#solana]

    A technology used in operating systems for
    programs that need to analyze network
    traffic, as is the case with smart
    contract technology.
    
    In a nutshell, it provides an interface to
    data link layers, allowing raw link-layer
    packets to be sent and received.
