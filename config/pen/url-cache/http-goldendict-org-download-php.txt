			<div class="post" id="release101">
				<h2 class="title">Latest release: GoldenDict 1.0.1</h2>
				<div class="entry">
					<p>New release. See <a href="changelog.php#release101">changelog</a> for details.<p />
          <ul>
          <li><a href="http://downloads.sourceforge.net/goldendict/GoldenDict-1.0.1-1-Install.exe">32-bit Windows installer</a>, 16MB - just run it to install the program. It doesn't include any local dictionaries, which are to be downloaded
          elsewhere. Wikipedia and websites would still work, though.</li>
          <li><a href="http://downloads.sourceforge.net/goldendict/GoldenDict-1.0.1-1-EnRuEn-Install.exe">32-bit Windows En-Ru-En installer</a>, 78MB -
	  this one in addition to the program itself has two dictionaries (Apresyan En-Ru and Smirnitskiy Ru-En) and English word pronunciations. It allows
	  trying out the program instantly without searching for any dictionary content. If you are upgrading from the previous version, do a
	  complete uninstall first, otherwise the program won't automatically use the content.</li>
          <li><a href="http://downloads.sourceforge.net/goldendict/GoldenDict-1.0.1-Portable.zip">32-bit Windows portable version</a>, 20MB -
	  just unzip it anywhere you want (e.g. pen drive) and put your dictionaries to the content/ directory.</li>
          <li><a href="http://downloads.sourceforge.net/goldendict/goldendict-1.0.1-src.tar.bz2">Source code</a>, 2.7MB - for Linux users and those who want to build the program themselves.
	  Consult <a href="buildfromgit.php">Building latest version from source</a><br> instructions for building hints. Linux users should better just use the means of their distribution to install the program.</li>
          <li><a href="http://downloads.sourceforge.net/goldendict/enruen-content-1.1.tar.bz2">En-Ru-En content</a>, 62MB - En-Ru-En content for Linux people, see description above.
	  Untar it to /usr/local/share/apps/goldendict. Remove any previous program configuration if you want the content to be hooked up automatically: do <span style="font-family: monospace;">rm -rf ~/.goldendict</span>.
	  </li>
   <li>Linux users: Linux versions usually come without morphology files. You can use system myspell/hunspell files,
   but there are better versions with more stems available specially for GoldenDict. Download them <a href="http://sourceforge.net/projects/goldendict/files/better%20morphologies/1.0/">here</a>.
   Unzip them to a separate directory and change morphology path in the program to point to it. Do not replace your system myspell dictionaries with those files - they don't add any new words, just more stems.
   </li>
   <li>If you need a good English dictionary with a clear and free license, get <a href="http://downloads.sourceforge.net/goldendict/WordNet3.0_1.0.zip">this one</a>, a full version of WordNet 3.0 specially formatted for GoldenDict.

          </ul>
