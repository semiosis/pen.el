As a name, Stack Overflow, is always written "Stack Overflow" (two words, capital letters). The website domain name is always written <strong>stackoverflow.com</strong> (no CamelCase, single word capitalization rules apply). Currently, all Stack Exchange Network sites follow this convention: Server Fault (serverfault.com), Super User (superuser.com), etc.</li>

                    <li class="my4">The <strong>Stack Exchange API </strong>allows users to write applications based on the Stack Exchange engine. The API is always referred to as "Stack Exchange API", even if the application is written for a specific site (i.e. never Stack Overflow API, Server Fault API, etc.).</li>
