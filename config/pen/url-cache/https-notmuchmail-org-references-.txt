<h2><a name="index1h2"></a>RFC</h2>

<p>Some of the most important RFCs related to Notmuch development are:</p>

<ul>
<li><p><a href="https://tools.ietf.org/html/rfc822">822</a> - Standard for the Format of Arpa
Internet Text Messages</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc1521">1521</a> - MIME (Multipurpose Internet Mail
Extensions) Part One: Mechanisms for Specifying and Describing the Format of
Internet Message Bodies</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc1847">1847</a> - Security Multiparts for MIME:
Multipart/Signed and Multipart/Encrypted</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc2047">2047</a> - Multipurpose Internet Mail
Extensions (MIME) Part Three: Message Header Extensions for Non-ASCII Text</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc2392">2392</a> - Content-ID and Message-ID
Uniform Resource Locators</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc2822">2822</a> - Internet Message Format</p></li>
<li><p><a href="https://tools.ietf.org/html/rfc3156">3156</a> - MIME Security with OpenPGP</p></li>
</ul>


<p>See the <a href="http://spruce.sourceforge.net/gmime/">GMime home page</a> for a more
comprehensive list of relevant RFCs.</p>
