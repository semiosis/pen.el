<h3 class='post-title entry-title' itemprop='name'>
Basic Haskell Examples
</h3>
<div class='post-header'>
<div class='post-header-line-1'></div>
</div>
<div class='post-body entry-content' id='post-body-225310375580393446' itemprop='description articleBody'>
<p>The Haskell community self-selects for people interested in unique things that Haskell can do that other languages cannot do. Consequently, a large chunk of Haskell example code in the wild uses advanced idioms (and I'm guilty of that, too).</p>
<p>So I would like to swing the pendulum in the other direction by just writing five small but useful programs without any imports, language extensions, or advanced features. These are programs that you could write in any other language and <strong>that's the point</strong>: you can use Haskell in the same way that you use other languages.</p>
<p>They won't be the prettiest or most type-safe programs, but that's okay.</p>
<h4 id="example-1-todo-program">Example #1: TODO program</h4>
<p>This program is an interactive TODO list:</p>
<pre class="sourceCode haskell"><code class="sourceCode haskell"><span class="ot">putTodo ::</span> (<span class="dt">Int</span>, <span class="dt">String</span>) <span class="ot">-&gt;</span> <span class="dt">IO</span> ()
putTodo (n, todo) <span class="fu">=</span> putStrLn (show n <span class="fu">++</span> <span class="st">&quot;: &quot;</span> <span class="fu">++</span> todo)

<span class="ot">prompt ::</span> [<span class="dt">String</span>] <span class="ot">-&gt;</span> <span class="dt">IO</span> ()
prompt todos <span class="fu">=</span> <span class="kw">do</span>
    putStrLn <span class="st">&quot;&quot;</span>
    putStrLn <span class="st">&quot;Current TODO list:&quot;</span>
    mapM_ putTodo (zip [<span class="dv">0</span><span class="fu">..</span>] todos)
    command <span class="ot">&lt;-</span> getLine
    interpret command todos

    <span class="kw">let</span> str12 <span class="fu">=</span> <span class="st">&quot;found them and &quot;</span>

    str13 <span class="ot">&lt;-</span> [<span class="st">&quot;saved the day.&quot;</span>, <span class="st">&quot;granted them three wishes.&quot;</span>]

    <span class="kw">let</span> str14 <span class="fu">=</span> <span class="st">&quot;  The end.&quot;</span>

    return (  str0
           <span class="fu">++</span> str1
           <span class="fu">++</span> str2
           <span class="fu">++</span> str3
           <span class="fu">++</span> str4
           <span class="fu">++</span> str5
           <span class="fu">++</span> str6
           <span class="fu">++</span> str7
           <span class="fu">++</span> str8
           <span class="fu">++</span> str9
           <span class="fu">++</span> str10
           <span class="fu">++</span> str11
           <span class="fu">++</span> str12
           <span class="fu">++</span> str13
           <span class="fu">++</span> str14
           )

main <span class="fu">=</span> <span class="kw">do</span>
    <span class="kw">let</span> len <span class="fu">=</span> length stories
    putStrLn (<span class="st">&quot;Enter a number from 0 to &quot;</span> <span class="fu">++</span> show (len <span class="fu">-</span> <span class="dv">1</span>))
    n <span class="ot">&lt;-</span> readLn
    putStrLn <span class="st">&quot;&quot;</span>
    putStrLn (stories <span class="fu">!!</span> n)</code></pre>
