         <h1 id="documentation">Documentation</h1>
<p>This page lists various resources to help you with Haskell. Resources marked with [$] require payment.</p>
<p>If you are new to Haskell and are not sure where to start from, we recommend <a href="https://www.seas.upenn.edu/~cis194/spring13/lectures.html">CIS194</a>. CIS194 is the introductory Haskell course of the University of Pennsylvania; it is free, thorough, practical and will guide you from the basics to advanced features of the language.</p>

<h2 id="library-documentation">Library Documentation</h2>
<p>Documentation for Haskell libraries is typically available on Hackage. We also have specialized tools for searching across it, not only by name, but by type.</p>
<ul>
<li><a href="http://www.haskell.org/hoogle/">Hoogle API Search</a></li>
<li><a href="http://hackage.haskell.org/">Hackage</a></li>
<li><a href="https://www.stackage.org">Stackage (with API Search)</a></li>
<li><a href="https://wiki.haskell.org/Typeclassopedia">The Typeclassopedia</a></li>
<li><a href="https://downloads.haskell.org/~ghc/latest/docs/html/libraries/index.html">Haddocks for Libraries included with GHC</a></li>
</ul>
<h2 id="language-report">Language Report</h2>
<p>The Haskell 2010 language report is available online as <a href="https://haskell.org/onlinereport/haskell2010/">HTML</a> and as <a href="https://haskell.org/definition/haskell2010.pdf">PDF</a>.</p>
<p>It can also be downloaded as a git repository:</p>
<pre><code>$ git clone -b h2010 https://github.com/haskell/haskell-report</code></pre>
<p>The differences between GHC and the report can be found <a href="http://www.haskell.org/ghc/docs/latest/html/users_guide/bugs.html#haskell-standards-vs-glasgow-haskell-language-non-compliance">in the GHC User’s Guide</a>.</p>
