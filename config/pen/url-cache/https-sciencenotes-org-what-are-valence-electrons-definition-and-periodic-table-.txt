class="wp-image-87549 lazyload"/><noscript><img  fetchpriority="high" decoding="async" width="1024" height="683"  src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" data-src="https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron-1024x683.jpg"  alt="Valence Electron" class="wp-image-87549 lazyload" data-srcset="https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron-1024x683.jpg 1024w, https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron-300x200.jpg 300w, https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron-150x100.jpg 150w, https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron-768x512.jpg 768w, https://sciencenotes.org/wp-content/uploads/2021/03/Valence-Electron.jpg 1500w" data-sizes="(max-width: 1024px) 100vw, 1024px" /></noscript></a><figcaption class="wp-element-caption">A valence electron is an outer shell electron that can participate in a chemical bond with another atom.</figcaption></figure><p>In chemistry and physics, a valence electron is an <a href="https://sciencenotes.org/what-is-an-electron/">electron</a> associated with an <a href="https://sciencenotes.org/10-interesting-atom-facts/">atom</a> that can form <a href="https://sciencenotes.org/types-of-chemical-bonds/">a chemical bond</a> and participate in a chemical reactions. Valence electrons are outer shell electrons for main group elements. For the <a href="https://sciencenotes.org/transition-metals/">transition metals</a> with partially-filed <em>d</em> shells, valence electrons are those electrons outside the noble gas core. The number of valence electrons indicates the maximum number of chemical bonds an atom can form.</p><h3 class="wp-block-heading">Number of Valence Electrons</h3><p>For main group elements, the number of valence electrons usually ranges between 1 and 8 because eight electrons forms a complete octet. Elements from groups have preferred numbers of valence electrons. For example, alkali metal atoms (e.g., lithium, sodium) have one valence electron. Alkaline earth atoms (e.g., magnesium, calcium) have two valence electrons. The noble gases have complete octets, so all eight of their electrons are valence electrons. The exception is helium, which has two valence electrons.</p><figure class="wp-block-image size-large"><a href="https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table.jpg" class="grouped_elements" rel="tc-fancybox-group87546"><img decoding="async" src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" data-src="https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-1024x683.jpg" alt="Valence Periodic Table" class="wp-image-87547 lazyload"/><noscript><img  decoding="async" width="1024" height="683"  src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" data-src="https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-1024x683.jpg"  alt="Valence Periodic Table" class="wp-image-87547 lazyload" data-srcset="https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-1024x683.jpg 1024w, https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-300x200.jpg 300w, https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-150x100.jpg 150w, https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table-768x512.jpg 768w, https://sciencenotes.org/wp-content/uploads/2021/03/valence-periodic-table.jpg 1500w" data-sizes="(max-width: 1024px) 100vw, 1024px" /></noscript></a><figcaption class="wp-element-caption">This periodic table shows the valences of element groups.</figcaption></figure><p>The transition metals make use of the <em>d</em>-subshell, which can accommodate 10 electrons. The <em>f</em>-subshell holds 14 electrons and the <em>g</em>-subshell contains up to 18 electrons. Metals in the middle of the periodic table become more stable by emptying a shell, half-filling it, or completely filling it. So, they can have more than 8 valence electrons.</p><h3 class="wp-block-heading">How to Find the Number of Valence Electrons</h3><p>The easiest way to find the number of valence electrons is to go by the element group in the valence periodic table. However, the most common method uses atom&#8217;s ground state <a href="https://sciencenotes.org/list-of-electron-configurations-of-elements/">electron configuration</a>. For main group elements, you&#8217;re looking for the number of electrons in the highest principal quantum number or the highest shell number. For example, in 1s<sub><sup>2</sup></sub><strong>2</strong>s<sup>2</sup> (helium), 2 is the highest quantum number. There are two 2s electrons, so a helium atom has two valence electrons. For transition metals, the number of valence electrons is the number of electrons in subshells past the atom&#8217;s noble gas core. For example, the electron configuration of scandium is [Ar]3d<sup>1</sup>4s<sup>2</sup>, for a total of 3 valence electrons.</p><div class='code-block code-block-3' style='margin: 8px auto; text-align: center; display: block; clear: both; height: 250px;'> <!-MONUMETRIC Repeatable 2 D:300x250 T:300x250 M:300x250,320x50 START-><div id="mmt-644db8f3-f7ce-4ecf-9c77-5115e6b4db87"></div> <!-MONUMETRIC Repeatable 2 D:300x250 T:300x250 M:300x250,320x50 ENDS-></div><h4 class="wp-block-heading">Examples</h4><ul><li>Magnesium&#8217;s ground state electron configuration is 1s<sup>2</sup>2s<sup>2</sup>p<sup>6</sup>3s<sup>2</sup>, the valence electrons would be the 3s electrons because 3 is the highest principal quantum number. Magnesium has two valence electrons.</li><li>Carbon&#8217;s ground state electron configuration is 1s<sup>2</sup>2s<sup>2</sup>2p<sup>2</sup>. The highest principal quantum number is 2. There are 2 electrons in the 2s subshell and 2 electrons in the 2 p subshell, giving carbon a total of four valence electrons.</li><li>Bromine&#8217;s ground state electron configuration is 1s<sup>2</sup>2s<sup>2</sup>p<sup>6</sup>3s<sup>2</sup>p<sup>6</sup>d<sup>10</sup>4s<sup>2</sup>4p<sup>5</sup>. The valence electrons are be the 4s and 4p electrons. Bromine has seven valence electrons.</li><li>The electron configuration of an iron atom is 1s<sup>2</sup>2s<sup>2</sup>2p<sup>6</sup>3s<sup>2</sup>3p<sup>6</sup>4s<sup>2</sup>3d<sup>6</sup>&nbsp;or&nbsp;[Ar]4s<sup>2</sup>3d<sup>6</sup>. Iron is a transition metal, so the number of valence electrons includes those in the 3d subshell, not just those in the 4s subshell. There are two electrons in the 4s subshell and 6 electrons in the 3d subshell, so iron has 8 valence electrons.</li></ul><h3 class="wp-block-heading">Valency vs Oxidation State</h3><p>Valency is the number of electrons in an atom&#8217;s outermost electron shell. <a href="https://sciencenotes.org/how-to-assign-oxidation-numbers/">Oxidation state</a> reflects the number of electrons that an atom actually can gain, lose, or share with another atom. The number of valence electrons indicates the maximum number of chemical bonds an atom can form, while oxidation state does not. Valency does not indicate electrical charge, while oxidation state does.</p><p>The number of valence electrons in an atom may have the same or different numerical value as its oxidation state. For example, a lithium atom has 1 valence electron and has an oxidation state of +1. In contrast, a neon atom has 8 valence electrons and an oxidation state of 0. A hydrogen atom has 1 valence electron. It has an oxidation state of +1 when it combines with most elements, but an oxidation state of -1 when it forms a compound with an alkali metal. The oxidation state of a pure element is always zero, but the number of valence electrons is not zero.</p><h3 class="wp-block-heading">References</h3><ul><li>IUPAC (1997). &#8220;Valence&#8221;.
