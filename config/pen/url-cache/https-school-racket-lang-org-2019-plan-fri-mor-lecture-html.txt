instance of the struct. meaning that if the variable isn&rsquo;t bound at all, we&rsquo;ll get
<span class="RktVal">#f</span> back (so we can raise our own error).</p><p><table cellspacing="0" cellpadding="0" class="RktBlk"><tr><td><span class="hspace">&nbsp;&nbsp;</span><span
