<h2><span id="Using_chemfig_to_draw_molecules_.28structural_formulae.29"></span><span class="mw-headline" id="Using_chemfig_to_draw_molecules_(structural_formulae)">Using <code>chemfig</code> to draw molecules (structural formulae)</span></h2>
<p>Drawing a molecule consists mainly of connecting groups of atoms with lines. Simple linear formulae can be easily drawn using the <code>chemfig</code> package, as shown in the following example:
</p>
<p class="mw-empty-elt"></p><div class="example">
<div class="code">
<div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\documentclass</span><span class="nb">{</span>article<span class="nb">}</span>
<span class="k">\usepackage</span><span class="nb">{</span>chemfig<span class="nb">}</span>
<span class="k">\begin</span><span class="nb">{</span>document<span class="nb">}</span>

<span class="k">\section</span><span class="nb">{</span>Introduction<span class="nb">}</span>
Writing chemical formulae with chemfig is straightforward.

<span class="k">\chemfig</span><span class="nb">{</span>A=B<span class="nb">}</span>
<span class="k">\end</span><span class="nb">{</span>document<span class="nb">}</span>
</pre></div>
</div>
</div>

<p><a class="open-in-overleaf" target="_blank" href="https://www.overleaf.com/docs?engine=pdflatex
&amp;snip_name=Chemfig+example+1
&amp;snip=%5Cdocumentclass%7Barticle%7D%0A%5Cusepackage%7Bchemfig%7D%0A%5Cbegin%7Bdocument%7D%0A%0A%5Csection%7BIntroduction%7D%0AWriting+chemical+formulae+with+chemfig+is+straightforward.%0A%0A%5Cchemfig%7BA%3DB%7D%0A%5Cend%7Bdocument%7D"><i class="fa fa-share"></i> Open this <code>chemfig</code> example in Overleaf</a>
</p><p class="mw-empty-elt"></p>
<p>The following image shows the output produced by the example above:
</p><p>

</p><p>The package is imported by <code>\usepackage{chemfig}</code> in the preamble. The command <code>\chemfig{A=B}</code> the draws the molecule. The symbol <code>=</code> determines the type of bond. See the <a href="#Reference_guide">reference guide</a> for a list of bond types.
</p>
<h3><span class="mw-headline" id="Angles">Angles</span></h3>
<p>There are several ways to define angles between bonds in molecules.
</p>
<p class="mw-empty-elt"></p><div class="example">
<div class="code">
<div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span>To define chemical formulae you can use units that define the angles

<span class="k">\chemfig</span><span class="nb">{</span>A-[1]B-[7]C<span class="nb">}</span>

Absolute angles

<span class="k">\chemfig</span><span class="nb">{</span>A-[:50]B-[:-25]C<span class="nb">}</span>

Relative angles

<span class="k">\chemfig</span><span class="nb">{</span>A-[::50]B-[::-25]C<span class="nb">}</span>
</pre></div>
</div>
</div>

</p><p class="mw-empty-elt"></p>

</p><p>Each one of the three commands in the example above uses a different method to determine the angle between bonds.
</p>
<ul><li><code>default units</code>: In the command <code>\chemfig{A-[1]B-[7]C}</code> the parameters inside brackets set the angle in special units, each unit equals 45°. Hence in the example the angles are 45° and 315°.</li>
<li><code>absolute units</code>: The angles can be set in absolute units, in the command <code>\chemfig{A-[:50]B-[:-25]C}</code> the parameter inside the brackets represent the angle, in degrees, measured from the horizontal baseline. Negative angles are allowed.</li>
<li><code>relative angles</code>: In the third example <code>\chemfig{A-[::50]B-[::-25]C}</code> the angles are measured from the previous bond, instead of the baseline.</li></ul>
<h3><span class="mw-headline" id="Rings">Rings</span></h3>
<p>The example below presents the syntax to draw regular polygons:
</p>
<p class="mw-empty-elt"></p><div class="example">
<div class="code">
<div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span>Regular polygons:

<span class="k">\chemfig</span><span class="nb">{</span>A*5(-B=C-D-E=)<span class="nb">}</span>

Incomplete rings are also possible:

<span class="k">\chemfig</span><span class="nb">{</span>A*5(-B=C-D)<span class="nb">}</span>
</pre></div>
</div>
</div>

</p><p>The syntax of the command <code>\chemfig{A*5(-B=C-D-E=)}</code> is explained below:
</p>
<ul><li><code>A</code>: This is the first atom, the rest of the atoms will be drawn from here</li>
<li><code>*5</code>: Number of sides of the polygon</li>
<li><code>(-B=C-D-E=)</code>: The rest of the atoms and bonds. If not enough are passed to complete the polygon, an incomplete ring will be drawn.</li></ul>

<h2><span class="mw-headline" id="Using_mhchem_to_typeset_chemical_formulae_and_equations">Using <code>mhchem</code> to typeset chemical formulae and equations</span></h2>
<p>The <code>mhchem</code> package’s versatile <code>\ce</code> command, which is <a href="#Visual_Editor’s_preview_of_chemical_formulae_and_equations">supported by Overleaf’s Visual Editor preview</a>, enables typesetting of chemical formulae and equations; for example, writing <code>\ce{H2O}</code> typesets \(\text{H}_2\text{O}\), the chemical formula for water.
</p>
<ul><li><strong>Note</strong>: Recent versions of the <code>mhchem</code> package can <a rel="nofollow" class="external text" href="https://www.overleaf.com/learn/how-to/Fixing_and_preventing_compile_timeouts#mhchemtimeout">increase the time needed for your project to compile</a>. Switching to the <a rel="nofollow" class="external text" href="https://ctan.org/pkg/chemformula?lang=en"><code>chemformula</code> package</a>, and using its <code>\ch</code> command, can decrease compilation time but the <code>\ch</code> command is not currently previewed by Visual Editor.</li></ul>
<h3><span class="mw-headline" id="How_to_load_the_mhchem_package">How to load the <code>mhchem</code> package</span></h3>
<p>To use the latest version of <code>mhchem</code>, currently version 4, add the following line to your document preamble: 
</p>
<p class="mw-empty-elt"></p><div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\usepackage</span><span class="na">[version=4]</span><span class="nb">{</span>mhchem<span class="nb">}</span>
</pre></div><p class="mw-empty-elt"></p>
<p>This loads <code>mhchem</code> with its default styles for formatting chemical formulae and equations. To change those defaults you can either use <code>mhchem</code> package options or the <code>\mhchemoptions</code> command. For example, to change the text mode font to sans serif and draw arrows using PGF/Ti<i>k</i>Z, you can load <code>mhchem</code> using              
</p>
<p class="mw-empty-elt"></p><div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\usepackage</span><span class="na">[version=4,arrows=pgf-filled,textfontname=sffamily]</span><span class="nb">{</span>mhchem<span class="nb">}</span>
</pre></div>
<p>or you can use the <code>\mhchemoptions</code> command within your document            
</p>
<p class="mw-empty-elt"></p><div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\mhchemoptions</span><span class="nb">{</span>arrows=pgf-filled,textfontname=sffamily<span class="nb">}</span>
</pre></div>
<p>The <a rel="nofollow" class="external text" href="http://mirrors.ctan.org/macros/latex/contrib/mhchem/mhchem.pdf"><code>mhchem</code> package documentation</a> provides further guidance on configuring <code>mhchem</code>, together with numerous examples of chemical formulae and equations, a few of which we reproduce here to help you get started. Note the use of space characters within more complex input to the <code>\ce</code> command. Those spaces are required to help <code>mhchem</code> correctly parse your chemical formulae and equations; omitting them can trigger the error <a href="#The_“Unexpected_input_character”_mhchem_error">Unexpected input character</a>.
</p>
<h3><span class="mw-headline" id="Some_examples">Some examples</span></h3>
<p>The following Overleaf project loads <code>mhchem</code> using three package options:  
</p>
<ul><li><code>arrows=pgf-filled</code>: draws filled arrows using PGF</li>
<li><code>textfontname=sffamily</code>: outside of math mode, chemical formulae and equations are typeset using using the default sans serif font</li>
<li><code>mathfontname=mathsf</code>: in math mode use a sans serif font</li></ul>
<p>The <code>\ce</code> command is used to typeset a range of examples.
</p>
<p class="mw-empty-elt"></p><div class="example">
<div class="code">
<div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\documentclass</span><span class="nb">{</span>article<span class="nb">}</span>
<span class="c">%Load mhchem using some package options</span>
<span class="k">\usepackage</span>[version=4,arrows=pgf-filled,
textfontname=sffamily,
mathfontname=mathsf]<span class="nb">{</span>mhchem<span class="nb">}</span>
<span class="k">\begin</span><span class="nb">{</span>document<span class="nb">}</span>
<span class="k">\subsection*</span><span class="nb">{</span>Simple molecules<span class="nb">}</span>

<span class="k">\begin</span><span class="nb">{</span>itemize<span class="nb">}</span>
<span class="k">\item</span> Water: <span class="k">\verb</span>|<span class="k">\ce</span><span class="nb">{</span>H2O<span class="nb">}</span>|, <span class="k">\ce</span><span class="nb">{</span>H2O<span class="nb">}</span>
<span class="k">\item</span> Benzene: <span class="k">\verb</span>|<span class="k">\ce</span><span class="nb">{</span>C6H6<span class="nb">}</span>|, <span class="k">\ce</span><span class="nb">{</span>C6H6<span class="nb">}</span>
<span class="k">\item</span> Hydrogen peroxide: <span class="k">\verb</span>|<span class="k">\ce</span><span class="nb">{</span>H2O2<span class="nb">}</span>|, <span class="k">\ce</span><span class="nb">{</span>H2O2<span class="nb">}</span>
<span class="k">\item</span> Acetic acid: <span class="k">\verb</span>|<span class="k">\ce</span><span class="nb">{</span>C2H4O2<span class="nb">}</span>|, <span class="k">\ce</span><span class="nb">{</span>C2H4O2<span class="nb">}</span>
<span class="k">\item</span> Glucose: <span class="k">\verb</span>|<span class="k">\ce</span><span class="nb">{</span>C6H12O6<span class="nb">}</span>|, <span class="k">\ce</span><span class="nb">{</span>C6H12O6<span class="nb">}</span>
<span class="k">\end</span><span class="nb">{</span>itemize<span class="nb">}</span>

<span class="k">\subsection*</span><span class="nb">{</span>Chemical equations<span class="nb">}</span>

Two basic examples:

<h3><span id="The_.E2.80.9CUnexpected_input_character.E2.80.9D_mhchem_error"></span><span class="mw-headline" id="The_“Unexpected_input_character”_mhchem_error">The “Unexpected input character” <code>mhchem</code> error</span></h3>
<p>The <code>mhchem</code> package error <strong><code>Assertion failed: Unexpected input character</code></strong> often results from missing spaces in the <code>\ce</code> command input—spaces are essential for <code>mhchem</code> to accurately parse the more complex chemical expressions. This error can arise when copying and pasting examples from PDF documents, where spaces may not be accurately replicated.
</p><p>
The following example demonstrates this error:</p><p class="mw-empty-elt"></p><div class="example">
<div class="code">
<div class="mw-highlight mw-highlight-lang-latex mw-content-ltr" dir="ltr"><pre><span></span><span class="k">\ce</span><span class="nb">{</span>CO2+C-&gt;2CO<span class="nb">}</span>
</pre></div>
</div>
</div>
